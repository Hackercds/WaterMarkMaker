# 自定义水印工具

一个纯前端的图片水印工具，支持**任意位置拖动**和**斜填充/平铺**两种水印模式，无需后端，直接在浏览器中完成水印添加并下载。

## 功能

- **单个水印模式**：在图片上任意拖动水印到想要的位置，支持 -180° 到 180° 旋转
- **斜填充/平铺模式**：网格平铺水印覆盖整张图片，可调节行数、列数、旋转角度和偏移
- **多种上传方式**：点击上传、拖拽上传、Ctrl+V 粘贴图片
- **文字自定义**：字体、字号、颜色、透明度均可调节
- **高清下载**：按原始分辨率渲染水印，导出 PNG 格式

## 使用方式

直接在浏览器中打开 `index.html` 即可使用。

或通过 GitHub Pages 访问：`https://hackercds.github.io/WaterMarkMaker/`

## 技术栈

纯 HTML + CSS + JavaScript（Canvas API），无任何依赖，无需构建。
